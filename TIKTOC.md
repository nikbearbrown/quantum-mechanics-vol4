## Concept
Volume 4 teaches the student to **use the density matrix, reason about entanglement and Bell tests, run basic quantum-information protocols, and read current quantum-hardware literature**, with the +1 simulation layer carried from Volume 1. Course textbook; chapter = week; read in sequence.

**Target course:** Senior elective / quantum-information bridge. 15-week map — chapters across Weeks 1–10, with the remaining weeks for the capstone project, review, and exam.

## The +1 Simulation System
Each chapter ends with an LLM Exercise block: the `CLAUDE.md` coding constitution, a copy-paste four-move D3 simulation prompt for the chapter's core object, exploration tasks, and an extension prompt. The Brutalist governing files are generated in Volume 1, Chapter 0 and travel with the student.

## Three-Act Arc
- Act One (1–3): the modern state — density matrix, composite systems, Bell.
- Act Two (4–6): information and openness — gates, teleportation, decoherence.
- Act Three (7–10): measurement, hardware, error, and research literacy.
